# Запуск раздела «Производство» на teamprof.ru

Готовый пакет для заливки. Стек: WordPress + Elementor + тема Flatsome.
Типографика и палитра уже выверены по сайту (Open Sans Condensed 700 — заголовки,
Ubuntu 500 — текст; акцент `#ffd54a`, тёмный `#363e4a`). Шаблон заскоуплен под
`#tp-production` и не влияет на остальной сайт.

Содержимое пакета:
- `images/stage1.jpg`, `stage2.jpg`, `stage3.jpg` — фото этапов (заливаются в Медиатеку);
- `production-elementor-template.json` — Elementor-шаблон страницы (PROD: пути к фото на teamprof.ru);
- `home-block.html` — разметка промо-бокса для главной.

---

## Шаг 1. Залить фото в Медиатеку

1. WordPress → **Медиафайлы → Добавить новый** → загрузить 3 файла из `images/`.
2. Проверить итоговые URL (Медиатека → файл → «Скопировать URL в буфер»). Ожидается:
   - `https://teamprof.ru/wp-content/uploads/2026/06/stage1.jpg`
   - `https://teamprof.ru/wp-content/uploads/2026/06/stage2.jpg`
   - `https://teamprof.ru/wp-content/uploads/2026/06/stage3.jpg`

   ⚠️ Если месяц/год загрузки другой, разбивка по датам отключена, или WP переименовал
   файл (добавил `-1` при совпадении имени) — **поправьте 3 URL** в шаблоне
   `production-elementor-template.json` (найти `2026/06/stage` и заменить на фактические),
   либо после импорта замените 3 ссылки прямо в Elementor (HTML-виджет, `<img src=...>`).

## Шаг 2. Импорт шаблона и создание страницы `/production/`

1. Elementor → **Шаблоны → Импорт шаблонов** → загрузить `production-elementor-template.json`
   (импортируется штатно).
2. **Страницы → Добавить новую**: заголовок **«Разработка и производство»**, slug **`production`**.
3. Шаблон страницы — **«Во всю ширину» (Full width)**, без сайдбара. Заголовок страницы скрыть
   (в шаблоне уже `hide_title: yes`; хедер/футер сайта остаются).
4. Редактировать в Elementor → «Мои шаблоны» → вставить импортированный шаблон → **Опубликовать**.

## Шаг 3. SEO

В Yoast/Rank Math для страницы продублировать (уже заданы в исходнике):
- **Title:** `Разработка и производство | TEAMPROF`
- **Description:** `TEAMPROF — российская торговая марка профессиональной иллюминации. Собственная
  инженерная разработка, контрактное производство под контролем TEAMPROF, собственные сборочные
  площади в России, контроль качества на каждом этапе, официальные поставки и маркировка
  «Честный ЗНАК».`
- **H1** на странице: «Разработка и PROизводство».

## Шаг 4. Промо-бокс на главной

1. Открыть главную в Elementor → ряд иконок-боксов `[Магазин] [О нас] [Наши клиенты] [Команда]`.
2. Скопировать image-box **«О нас»** и вставить **сразу после него** (перед «Наши клиенты»).
   Либо вставить готовую разметку из `home-block.html` как HTML-виджет в ту же позицию.
3. В новом боксе: ссылка → `https://teamprof.ru/production/`; заголовок →
   **«СОЗДАЁМ ПРОДУКТ, А НЕ ПЕРЕПРОДАЁМ»**; описание → **«УЗНАТЬ О ПРОИЗВОДСТВЕ TEAMPROF»**.
4. Итоговый порядок: `[Магазин] [О нас] [►Производство] [Наши клиенты] [Команда]`.

   ⚠️ **Иконка промо-бокса** — единственный недоделанный ассет: нужна монохромная иконка
   «производство» 150×150 px в стиле `ик2/ик3/ик4` (из `/wp-content/uploads/2020/06/`).
   Сейчас в `home-block.html` подставлен плейсхолдер `ик4`. Заменить `src` на фирменную иконку,
   когда будет готова.

## Шаг 5. Пункт меню «Производство»

Внешний вид → Меню (верхнее меню):
1. Добавить пункт типа «Страница» → «Разработка и производство» (или «Произвольная ссылка» → `/production/`).
2. Подпись: **«Производство»**.
3. Перетащить **между «О нас» и «Наши клиенты»**. Сохранить.

## Шаг 6. Приёмка (чек-лист)

- [ ] `/production/` открывается, стиль совпадает с сайтом, на мобильном блоки этапов
      складываются корректно (текст → фото).
- [ ] 3 фото этапов отображаются (не «битые» ссылки), показаны целиком.
- [ ] Промо-бокс строго между «О нас» и «Наши клиенты»; ведёт на `/production/`.
- [ ] Пункт меню «Производство» на нужной позиции.
- [ ] Заданы Title / Description / H1.
- [ ] (после готовности) заменена иконка промо-бокса.
