Раздел «Производство» — пакет для внедрения

Готовая страница /production/ для teamprof.ru (WordPress + Elementor + Flatsome). Ниже — превью, файлы пакета и пошаговая инструкция по внедрению.

▶ Открыть превью страницы ⬇ Скачать пакет (.zip)

Файлы пакета

  • teamprof-production-package.zip всё одним архивом: шаблон + 3 фото + промо-бокс + инструкция
  • production-elementor-template.json Elementor-шаблон страницы (пути к фото уже на teamprof.ru/wp-content/uploads/2026/06/)
  • images/stage1.jpg · stage2.jpg · stage3.jpg фото этапов — залить в Медиатеку WordPress
  • home-block.html разметка промо-бокса «Производство» для главной
  • ЗАПУСК.md тот же чек-лист, что ниже, — текстом

Как внедрить

  1. Залить фото в Медиатеку

    WordPress → Медиафайлы → Добавить новый → загрузить 3 файла (stage1/2/3.jpg). Ожидаемые URL: teamprof.ru/wp-content/uploads/2026/06/stageN.jpg.

  2. Импорт шаблона и страница /production/

    • Elementor → Шаблоны → Импорт → production-elementor-template.json.
    • Создать страницу «Разработка и производство», slug production, шаблон «Во всю ширину», заголовок скрыт.
    • Вставить импортированный шаблон → Опубликовать.

  3. SEO

    Title: Разработка и производство | TEAMPROF; H1: «Разработка и PROизводство». Description — в файле ЗАПУСК.md.

  4. Промо-бокс на главной

    Вставить home-block.html (или копию бокса «О нас») между «О нас» и «Наши клиенты». Ссылка → /production/, заголовок «СОЗДАЁМ ПРОДУКТ, А НЕ ПЕРЕПРОДАЁМ».

  5. Пункт меню

    Внешний вид → Меню → добавить пункт «Производство» (→ /production/), поставить между «О нас» и «Наши клиенты».
2 момента при заливке:
  • URL фото в шаблоне проставлены под папку 2026/06. Если месяц/год другой или WP переименовал файл — поправить 3 ссылки (в шаблоне или в Elementor, <img src>).
  • Иконка промо-бокса — пока плейсхолдер ик4. Нужна монохромная иконка «производство» 150×150 в стиле ик2/ик3/ик4.
Важно по формулировкам: модель — контрактное (OEM/ODM) производство под контролем TEAMPROF + собственные сборочные площади в РФ. Недопустимы формулировки о «собственном заводе полного цикла в России». Текст в шаблоне уже выверен — не редактировать без согласования.
